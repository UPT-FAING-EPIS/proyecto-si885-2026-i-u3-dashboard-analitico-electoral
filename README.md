# Dashboard Analitico Electoral

## Descripcion del sistema

El Dashboard Analitico Electoral es un sistema de inteligencia de datos orientado al analisis de informacion politica y electoral. Su objetivo es centralizar, organizar y visualizar datos relevantes sobre candidatos, partidos politicos, propuestas, financiamiento de campana, visitas electorales, trayectoria profesional, formacion academica, redes sociales y antecedentes registrados en fuentes oficiales.

El sistema integra procesos de extraccion, transformacion y carga de datos, una base de datos estructurada y un tablero desarrollado en Power BI. A traves de sus modulos, permite comparar candidatos, revisar propuestas por tema, analizar ingresos y gastos de campana, consultar informacion de perfiles politicos y apoyar la toma de decisiones mediante visualizaciones claras e interactivas.

## Modulos principales

- Perfil del candidato.
- Comparador de propuestas electorales.
- Financiamiento por campana electoral.
- Visitas de campana.
- Analisis de informacion academica, laboral, politica y patrimonial.
- Integracion de datos desde fuentes oficiales y archivos estructurados.

## Estructura general

- `sistema/data`: archivos de datos utilizados por el sistema.
- `sistema/database`: scripts SQL, vistas y consultas para el modelo de datos.
- `sistema/dashboard`: archivos del dashboard y modelo semantico de Power BI.
- `sistema/scripts`: scripts de extraccion, procesamiento y exportacion de informacion.
- `sistema/docs`: documentacion tecnica de conexion, despliegue y automatizacion.
- `Documentacion`: informes, presentaciones y evidencias del proyecto.
